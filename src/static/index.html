Single Layer

POST: /api/single-layer/network
BODY (JSON):
edge_list - edge list of network in format: from to
layout - one of following: spring | circular | spiral
Example: 
{
    "edge_list": "0 1\n0 2\n2 1",
    "layout": "spring"
}
POST: /api/single-layer/community
BODY (JSON):
edge_list - edge list of network in format: from to
community_list - edge list of network in format: actor community
layout - one of following: spring | circular | spiral
Example: 
{
    "edge_list": "0 1\n0 2\n1 2",
    "community_list": "0 1\n1 2\n2 2",
    "layout": "spring"
}

Multi Layer

POST: /api/multi-layer/diagonal
BODY (JSON):
edge_list - edge list of network in format: from layerFrom to layerTo weight
Example: 
{
    "edge_list": "0 l1 1 l1 1\n0 l2 2 l2 2\n1 l1 2 l2 3"
}
POST: /api/multi-layer/hairball
BODY (JSON):
edge_list - edge list of network in format: from layerFrom to layerTo weight
community_list - edge list of network in format: actor community
Example: 
{
    "edge_list": "0 l1 1 l1 1\n0 l2 2 l2 2\n1 l1 2 l2 3",
    "community_list": "0 1\n1 2\n2 2"
}

Common Charts

POST: /api/common-charts/barplot
BODY (JSON):
X - categories
Y - values
labels - X item labels
xlabel - label for x axis
ylxabel - label for y axis
Example: 
{
    "X": [1, 2, 3],
    "Y": [1, 2, 3],
    "labels": [1, 2, 3],
    "xlabel": "X Data",
    "ylabel": "Y Data"
}
POST: /api/common-charts/barplot
BODY (JSON):
sizes - sizes of items
labels - labels for items
Example: 
{
    "sizes": [1, 2, 3],
    "label": ["a", "b", "c"]
}
